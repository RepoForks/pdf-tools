PDF Tools
=========

A set of tools for PDF files.

.. image:: https://travis-ci.org/arminha/pdf-tools.svg?branch=master
   :target: https://travis-ci.org/arminha/pdf-tools

Features
--------

- Change permissions (print, copy etc)
- Combine PDF files or individial pages from PDF files.

Download
--------

Get the `latest release <https://github.com/arminha/pdf-tools/releases/latest>`__.
